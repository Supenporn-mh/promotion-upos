# Release Notes: Promotion-UPOS-new (v1.0.0)
**Date:** 2026-03-11  
**Approved By:** Agent 9 (Release Manager)  

## 🎉 Summary
ระบบ Promotion Management (Plug-in) สำหรับ UPOS พร้อมสำหรับการทดสอบในระดับ UI และ Mock-API แล้ว ระบบนี้ถูกออกแบบมาแบบ Mobile-First เพื่อให้ Cashier และ Admin ที่ใช้เครื่อง POS หรือแท็บเล็ตสามารถกดเปิด Tab ใหม่ขึ้นมาตั้งค่าโปรโมชั่นได้อย่างสะดวก

## 🛠 Features Included
- **[Feature]** หน้า Dashboard แสดงรายการโปรโมชั่นทั้งหมด พร้อมรองรับ Pagination และ Search Box เรียลไทม์
- **[Feature]** หน้า "+ เพิ่มโปรโมชั่น" รองรับการกำหนดเงื่อนไขโปรโมชั่น 3 ประเภท:
  - ส่วนลด (เปอร์เซ็นต์/จำนวนเงิน)
  - แลกซื้อ
  - ของแถม
- **[Feature]** รองรับการซิงก์รหัสสิทธิ (สมาชิก / ลูกค้าทั่วไป)
- **[Feature]** ส่วนต่อขยาย API `GET /api/v1/pos/products` สำหรับดึงข้อมูลสินค้าในบิลของ POS มาผูกเข้ากับเงื่อนไขโปรโมชั่น

## 📡 Technical Stack
- **Frontend Stack**: Vue 3 (Option API), Vite, Tailwind CSS, Pinia, Axios
- **Icon Set**: Phosphor Icons (Vue version)
- **Mock Server**: Express.js

## 🛑 Known Limitations (For Next Iteration)
